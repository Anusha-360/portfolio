Indicwiki

Indicwiki is an internship based on wikipedia articles. Wikipedia is perhaps the most widely used resource of the Encyclopaedic Knowledge, Education and E-Literacy platform on the Web. It is a free, online encyclopaedia with over 7 million articles only in English. India is in second place with 117 million views per day behind the USA 189 million views as of January 2019. The goal of this internship is to generate approximately 10k wiki articles in telugu language given a particular domain of interest. Each wiki article generated should meet a minimum word count of 500 words and also have references, infobox, categories, image(optional) etc.

Sprint-1

Gone through student training modules about wikipedia articles. Collected attributes for the cities by referring to the wikipedia article.

Sprint-2

Based on the introductory paragraph of the wikipedia article about the cities taken from the “World Best Cities” website we identified the attributes for which we need to extract data. Translated all the data in the unified spreadsheet from English to Telugu using “GOOGLE TRANSLATE”. Collected all the sentences in a google spreadsheet and also listed out the attributes with the data type.

Sprint-3

Created telugu titles for 10K cities. Translated the city names using google translate in a google spreadsheet and identified the mistakes made by google translate and rectified them. Collected data for finalised introductory attributes by referring to wolf from alpha and simple maps. Were unable to translate the infoboxes labels from english to telugu that are generated for the cities wikipedia articles.

Sprint-4

Add randomization and categories to the jinja Template. Create sample articles for different types of cities in different countries/ continents. Collected all the sentences in a google spreadsheet and also listed out the attributes with the data type.

Sprint-5

Collected the data for all the given attributes of domain “demographics”. Collected the data and wiki links for all the cities in the english wikipedia. Collected the references for all the attributes. Created sample articles for different types of cities in different continents. Worked on adding randomization and categories of wikipedia articles to the template generated .

Sprint-6

Collected the data for all the given attributes of respective sub-domain. Cleaned the dataset for handling missing values. Worked on all the 21000 cities collected . Worked on dbpedia queries to generate wikipedia data attributes for demographics subdomain.

Sprint-7

Added more 12,000 cities for the data set generated in previous sprint, final count of unified cities 29,055. Collected all the demographics attributes for the cities that were newly added Collected wiki links and data of subdomain for 12k cities. Worked on generating template for demographics subdomain.

Sprint-8
